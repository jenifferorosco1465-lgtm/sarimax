# Publicación en GitHub y Vercel

1. Cree un repositorio vacío en GitHub.
2. Ejecute los comandos del README.
3. Verifique que GitHub Actions finalice en verde.
4. Importe el repositorio en Vercel.
5. Configure `dashboard` como Root Directory y framework Next.js.
6. No configure secretos estadísticos en el frontend.
7. Cada nueva corrida debe actualizar `dashboard/public/data/*.json`, hacer commit y activar un nuevo despliegue.
